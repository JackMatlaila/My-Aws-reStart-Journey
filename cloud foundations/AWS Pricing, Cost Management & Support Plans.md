# AWS Financial Management & Support Plans

## 1. AWS Pricing Concepts
- **Pay for what you use:** No long-term contracts or complex licensing.  
- **Reserved Instances:** Discounted rates for 1–3 year commitments.  
- **Volume Discounts:** Lower per-unit cost with increased usage.  
- **Free Tier:** 12-month free access to services like EC2, S3, RDS, IAM.

## 2. TCO (Total Cost of Ownership) Tools
- **AWS Pricing Calculator:** Estimates monthly service costs.  
- **AWS Migration Evaluator:** Analyzes on-prem data to suggest optimal AWS setups.

## 3. Cost Management & Optimization
- **AWS Budgets:** Set alerts for cost thresholds.  
- **AWS Cost Explorer:** Analyze and forecast costs.  
- **Cost Allocation Tags & CUR Reports:** Organize and track spending.  
- **Savings Tools:**  
  - Cost Optimization Hub  
  - Savings Plans (commit usage for lower rates)  
  - Reserved Instances (EC2, RDS, Redshift, DynamoDB)

## 4. Billing & Account Management
- **Consolidated Billing:** Manage multiple accounts under one payment.  
- **AWS Organizations:** Centralized account management and governance.  
- **Volume Discounts:** Save more with higher usage.

## 5. AWS Support Plans
| Plan | Key Features |
|------|---------------|
| **Basic** | Free access to docs, forums, 7 Core Trusted Advisor checks |
| **Developer** | Best-practice guidance, email support, 7 Core checks |
| **Business** | 24×7 phone/chat/email, all Trusted Advisor checks, 3rd-party support |
| **Enterprise On-Ramp** | TAM pool, Concierge team, Infrastructure Event Manager |
| **Enterprise** | Dedicated TAM, 15-min critical response, operations reviews |

## 6. Technical Account Manager (TAM)
Guides Enterprise customers with architecture reviews, cost optimization, and access to AWS experts.
